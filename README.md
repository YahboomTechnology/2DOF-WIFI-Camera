# Introduction
This camera platform is mainly composed of two 180Â° servos, a WIFI camera and a set of acrylic bracket. The camera comes with a PH2.0-4PIN anti-reverse connection interface, adopts serial communication, and only needs 5V power to work normally. Users can connect to the WIFI hotspot that comes with the camera to form a local area network between the mobile phone and the MCU(robot car) to realize video transmission and remote control.
